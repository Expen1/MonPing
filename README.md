# MonPing
Professional Network Monitoring &amp; Research Tool

**MonPing** is a professional network monitoring and research tool designed for researchers, network analysts, and students.

It helps study internet connectivity patterns, DNS resolution behavior, IP reachability, and filtering dynamics across different providers and countries.

---

### Features

- Real-time ping monitoring with ICMP + TCP fallback
- DNS resolver testing (response time & availability)
- Support for CIDR ranges (automatically samples IPs)
- Quick provider loading (Cloudflare, AWS, Google, Azure, etc.)
- Stealth mode to reduce detection
- Traceroute analyzer
- Export results as JSON or CSV for research
- Clean and fast dark UI

### Usage

1. Run `MonPing.exe`
2. Click **"Start Full Research Scan"**
3. Use **Stealth Mode** for more natural behavior
4. Load popular providers (Cloudflare, AWS, etc.) with one click
5. Export data for academic/research use
