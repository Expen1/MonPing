# MonPing - Professional Network Monitoring & Research Tool

MonPing is a powerful, professional network monitoring tool designed for researchers, network analysts, students, and those studying internet infrastructure and filtering dynamics.

It helps analyze connectivity patterns, DNS behavior, IP reachability, and censorship circumvention possibilities across different providers.

### ✨ Key Features

- **Real-time Multi-Layer Monitoring**: ICMP Ping + Multi-Port TCP Connect + HTTP Fallback
- **Advanced AI Engine**: EWMA Predictive Latency, Variance Analysis, Flap Detection
- **Health Score** (0-100) for each target
- **Trend Analysis**: Improving / Stable / Degrading detection
- **Auto Blacklist** for low-quality and unstable paths
- **Full CIDR Support** with intelligent random IP sampling
- **Expanded Cloudflare IP Ranges** (very comprehensive)
- **DNS Resolver Testing** with multiple providers (Irancell, MCI, etc.)
- **One-Click Provider Loading**: Cloudflare, AWS, Google, Azure, Major Ranges
- **Stealth Mode** to reduce detection risk
- **Live Latency Trend Graph**
- **Bypass Assistant** tab with smart recommendations for filtering circumvention
- **Traceroute Analyzer** tool
- **Right-click Context Menu** on targets (Traceroute, Copy IP, Blacklist, Remove)
- **Professional Dark UI** optimized for long scanning sessions
- **Pause / Resume** functionality
- **Export Data** as JSON or CSV for research purposes
- **Import / Export** target lists

### Usage

1. Run `MonPing.exe`
2. Click **"START SCAN"** to begin monitoring
3. Enable **Stealth Mode** for quieter operation
4. Use the **Load** button to add major providers (especially Cloudflare)
5. Check the **Bypass Assistant** tab for the best working paths
6. Use right-click on any target for more options
7. Export results anytime for research or analysis
